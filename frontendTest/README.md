# Frontend Test Files

Place your test files here:
- `test_vocal.wav` - Your extracted vocal audio file (you need to add this)
- `lyrics.txt` - Test lyrics (already created, you can edit this)

## Usage:
1. Replace or add `test_vocal.wav` with your actual test vocal file
2. Edit `lyrics.txt` with your test lyrics if needed
3. Click "Use Test Files" button in frontend to skip vocal extraction

## Notes:
- The test_vocal.wav file should be a clean vocal track (already extracted/separated)
- Edit lyrics.txt to match your test audio
- This saves time during development by skipping the Demucs vocal extraction step
